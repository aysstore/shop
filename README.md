# 🛍️ فروشگاه اینترنتی آیس (Ays)

یک فروشگاه اینترنتی کامل، امن و مدرن با معماری MERN Stack (MongoDB, Express.js, React, Node.js).

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Node.js](https://img.shields.io/badge/Node.js-18.x-green.svg)](https://nodejs.org/)
[![React](https://img.shields.io/badge/React-18.x-blue.svg)](https://reactjs.org/)

---

## ✨ ویژگی‌ها

### 🛡️ امنیت
- 🔐 احراز هویت امن با JWT و Refresh Token
- 🛡️ محافظت در برابر حملات XSS و CSRF
- 🔒 رمزنگاری اطلاعات حساس
- 🚦 محدودیت نرخ درخواست (Rate Limiting)
- ✅ اعتبارسنجی کامل ورودی‌ها

### 🛒 فروشگاه
- 📦 مدیریت کامل محصولات (افزودن، ویرایش، حذف)
- 🏷️ دسته‌بندی محصولات با سلسله‌مراتب
- 🔍 جستجوی پیشرفته با فیلترهای متنوع
- ⭐ سیستم امتیازدهی و نظرات
- 🎯 محصولات ویژه، جدید و پرفروش

### 🛍️ سبد خرید و پرداخت
- 🛒 سبد خرید پیشرفته با مدیریت موجودی
- 💳 اتصال به درگاه‌های پرداخت (Stripe, Zarinpal)
- 🚚 محاسبه خودکار هزینه ارسال
- 📦 پیگیری سفارش با کد رهگیری

### 👤 پنل کاربری
- 📊 داشبورد مدیریت سفارشات
- ❤️ لیست علاقه‌مندی‌ها
- 📝 ویرایش پروفایل و تغییر رمز
- 📍 مدیریت آدرس‌های ارسال

### 🔧 مدیریت (ادمین)
- 📊 داشبورد مدیریت کامل
- 👥 مدیریت کاربران
- 📦 مدیریت سفارشات
- 🏷️ مدیریت دسته‌بندی‌ها
- 📈 گزارش‌های فروش
